# FocusCity - デザイン設計書

## アプリコンセプト
スマホを触らなかった時間・勉強・仕事への集中時間に応じて資源を獲得し、その資源を使って自分だけの街を育てるシムシティ風ゲームアプリ。

## 画面一覧

### 1. ホーム画面（街マップ）
- **メイン画面**：グリッドベースの街マップ（8×12グリッド）
- スクロール・ピンチズーム対応
- 各グリッドセルに建物・道路・畑を配置
- 右上：資源ステータスバー（木材・鉄・農作物・コイン）
- 下部：建設メニューボタン
- 左上：フォーカスタイマーボタン（集中モード開始）

### 2. 資源・報酬画面
- 現在保有資源の一覧（木材・鉄・農作物の種・道路材料・コイン）
- オフライン報酬の受け取りUI（スマホを離れていた時間の計算）
- 資源の説明・用途

### 3. 建設メニュー画面（モーダル）
- 建設可能な建物一覧
- 必要資源の表示
- カテゴリ：住宅・商業・農業・工業・インフラ

### 4. フォーカスタイマー画面
- 集中モードのタイマー（25分ポモドーロ or カスタム）
- タイマー中は大きなカウントダウン表示
- 終了時に獲得資源のアニメーション表示
- 集中記録の履歴

### 5. 建物詳細画面（モーダル）
- 建物情報・レベル
- 生産状況（工場・畑）
- アップグレードオプション

### 6. 進捗・実績画面
- 集中時間の累計
- 建設した建物の数
- 街の発展レベル
- バッジ・実績

## 主要コンテンツ・機能

### 資源システム
| 資源 | 獲得方法 | 用途 |
|------|---------|------|
| 木材 | オフライン時間・フォーカス | 家・柵・農場建設 |
| 鉄 | フォーカス時間・工場 | 工場・道路・高層建物 |
| 農作物の種 | オフライン時間 | 畑の設置 |
| 道路材料 | フォーカス時間 | 道路建設 |
| コイン | 全般・建物収益 | 建物アップグレード |
| 部品 | 工場生産 | 高度な建物 |
| 農作物 | 畑生産 | コイン変換・食料 |

### 建物カテゴリ
| カテゴリ | 建物例 | 必要資源 |
|---------|--------|---------|
| 住宅 | 小屋・家・マンション | 木材・鉄 |
| 農業 | 畑・温室・農場 | 木材・農作物の種 |
| 工業 | 工場・製材所・鉄工所 | 鉄・道路材料 |
| 商業 | 市場・店舗・銀行 | 木材・鉄・コイン |
| インフラ | 道路・橋・公園 | 道路材料・木材 |

## キーユーザーフロー

### フロー1：初回起動
1. スプラッシュ → ホーム（空の平地マップ）
2. チュートリアル吹き出し「スマホを置いて集中しよう！」
3. フォーカスタイマーを開始 → 資源獲得 → 最初の建物を建設

### フロー2：オフライン報酬受け取り
1. アプリ起動 → 「おかえり！X時間X分離れていましたね」
2. 獲得資源のアニメーション表示
3. 資源をタップして受け取り → ホームに戻る

### フロー3：建物建設
1. ホームの「＋建設」ボタンタップ
2. 建設メニュー表示 → カテゴリ選択
3. 建物選択 → 必要資源確認 → 「建設」
4. マップ上の空きセルをタップ → 建物配置
5. 建設アニメーション → 完了

### フロー4：工場・畑の生産
1. 配置済み工場/畑をタップ
2. 生産状況確認（進行バー）
3. 完了した生産物を「収穫」タップ
4. 資源に追加

### フロー5：フォーカスタイマー
1. 左上のタイマーボタンタップ
2. 集中時間を設定（25分 / 50分 / カスタム）
3. タイマー開始 → カウントダウン表示
4. 完了 → 獲得資源アニメーション
5. 街マップに戻る

## レイアウト方針
- **ポートレート固定**（9:16比率）
- タブバー：下部（街マップ・資源・フォーカス・進捗）
- グリッドマップ：スクロール可能、タップで建設/詳細
- モーダルシート：建設メニュー・建物詳細
- 資源バー：常時上部に固定表示
